Інструкція користувача
Запуск сервісу:

a. Локальний запуск:
- Встановіть залежності за допомогою poetry install.
- Запустіть сервер FastAPI за допомогою poetry run uvicorn app.main:app --reload.
- Перейдіть до http://127.0.0.1:8000/docs у вашому браузері для взаємодії з API.

b. Запуск за допомогою Docker:
- Побудуйте Docker образ за допомогою docker build -t image_recognition_app ..
- Запустіть Docker контейнер за допомогою docker run -p 8000:8000 image_recognition_app.
- Перейдіть до http://127.0.0.1:8000/docs у вашому браузері.

Використання API:

Перейдіть до документації API http://127.0.0.1:8000/docs.
@@@@@@@

Автори та ліцензія:

Перевірте розділи "Автори" та "Ліцензія" в файлі README.md для отримання додаткової інформації.
Завдяки цим інструкціям ви зможете запустити і використовувати сервіс для розпізнавання та категоризації зображень.
